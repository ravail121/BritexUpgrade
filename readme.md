## BriteX Backend

This repository includes backend code for BriteX


### Configuration Steps

Follow given steps to run:

- Clone repository
- `composer install`
- `cp .env.example .env`
- Put appropriate configuration in .env
- `php artisan migrate`
- `php artisan serve`
